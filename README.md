# Magnavita Serviços Marítimos ⚓

Sistema profissional de gestão e portal de serviços náuticos. Excelência em habilitações, certificados e consultoria marítima.

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido com tecnologias modernas para garantir performance e segurança:

- **Vite** & **TypeScript** - Build rápido e código tipado.
- **React** - Interface dinâmica e responsiva.
- **Supabase** - Banco de Dados, Autenticação e Realtime.
- **Tailwind CSS** & **shadcn/ui** - Design premium e componentes modernos.
- **Lucide React** - Ícones profissionais.

## 🛠️ Como rodar o projeto localmente

1. **Clone o repositório**
2. **Instale as dependências**

   ```sh
   npm install
   ```

3. **Inicie o servidor de desenvolvimento**

   ```sh
   npm run dev
   ```

## 📋 Funcionalidades Principais

- **Portal de Serviços**: Exibição dinâmica de serviços náuticos.
- **Agendamento Inteligente**: Formulários com preenchimento automático de endereço (ViaCEP).
- **Gerador de Documentos**: Geração automática de procurações em .docx.
- **Painel Administrativo Próprio**: Gestão de clientes, serviços e avaliações em tempo real.
- **CRM Integrado**: Busca avançada e controle de processos.
